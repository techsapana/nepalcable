## Contact Form Email Setup

To send contact form submissions to `necables@hotmail.com`, add these values to your `.env`:

```env
CONTACT_SMTP_HOST=smtp.office365.com
CONTACT_SMTP_PORT=587
CONTACT_SMTP_USER=necables@hotmail.com
CONTACT_SMTP_PASS=YOUR_HOTMAIL_PASSWORD_OR_APP_PASSWORD
CONTACT_RECEIVER_EMAIL=necables@hotmail.com
```

Notes:

- Use an app password if your Microsoft account has 2FA enabled.
- The form submits to `POST /api/contact` and the server sends the email using SMTP.
